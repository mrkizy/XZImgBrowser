# XZImgBrowser
一款仿微博的图片浏览器工具<br>
<b>使用方法：</b><br>
1:下载项目，直接将“图片浏览器”文件夹里文件拖到自己的项目里即可使用。<br>
2:调用方法showViewWithArray:(NSArray *)imgArr beginWithIndex:(NSInteger)index即可实现图片浏览<br>
第一个参数传递需要浏览的图片(UIImageView)数组，第二个参数表示默认第几张图片开始浏览。<br><br>
